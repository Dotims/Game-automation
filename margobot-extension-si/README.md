# MargoSzpont - Browser Extension

## Instalacja (Chrome / Brave / Opera GX / Edge)

### 1. Otwórz stronę rozszerzeń
- **Chrome/Brave:** wpisz w pasku adresu `chrome://extensions`
- **Opera GX:** wpisz `opera://extensions`
- **Edge:** wpisz `edge://extensions`

### 2. Włącz tryb deweloperski
- Znajdź przełącznik "Developer mode" / "Tryb deweloperski" (zwykle w prawym górnym rogu)
- Włącz go

### 3. Załaduj wtyczkę
- Kliknij "Load unpacked" / "Załaduj rozpakowane"
- Wybierz folder `margobot-extension`

### 4. Gotowe!
- Ikona kota 🐱 pojawi się na pasku narzędzi
- Kliknij na nią aby otworzyć panel

---

## Użycie

1. **Otwórz grę Margonem** i zaloguj się na postać
2. **Kliknij ikonę wtyczki** (kot) na pasku przeglądarki
3. **Wpisz klucz licencji** i kliknij "Aktywuj"
4. Kliknij **"Uruchom Bota"** - panel bota pojawi się w grze
5. Ustaw poziomy mobów i kliknij **START**

---

## Funkcje

- ⚔️ Automatyczne atakowanie mobów w zadanym przedziale poziomów
- 💊 Auto-heal (automatyczne używanie mikstur)
- 🎮 Panel UI wstrzyknięty do gry
- 📦 Działa na: Chrome, Brave, Opera GX, Edge

---

## Uwagi

- Wtyczka wymaga dostępu do `margonem.pl`
- Bot działa tylko gdy jesteś zalogowany w grze
- Aby zatrzymać bota: kliknij STOP w panelu lub w popup wtyczki
